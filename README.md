# IoT-Home-Automation-System
This project simulates a smart home automation system using sensors to control lighting and detect human presence. It demonstrates the application of IoT concepts in creating intelligent and responsive environments.

Key Features

Automatic lighting system using PIR sensor and LDR

Motion and distance detection for smart door system

Simulation of real-world IoT automation scenarios

Tech Stack

Tinkercad (IoT simulation)

Sensors: PIR, LDR

Embedded system logic

How it Works

PIR sensor detects human movement

LDR measures ambient light intensity

Lights automatically turn ON/OFF based on conditions

Door system detects presence and distance

Results

Successfully simulated an automated smart home environment

Demonstrated efficient use of sensors for real-time control
